# EduFlow AI - Site Vitrine

## 🎓 Présentation du projet

**EduFlow AI** est un système intelligent de gestion de cursus académique conçu pour les étudiants de l'ISI (Institut Supérieur d'Informatique). Il répond à la problématique : *"Comment automatiser la priorisation des tâches en fonction du poids réel de chaque matière dans le diplôme ?"*

Ce dépôt contient la **version vitrine** du projet, livrée dans le cadre de l'atelier créatif (Samba B. Kane, L2 IAGE).

## 🚀 Fonctionnalités présentées

- ✅ Module LMD avec gestion des 30 crédits ECTS
- ✅ Algorithme IA de priorisation : `Priorité = (Crédits × Coefficient) / Jours restants`
- ✅ Tableau de bord avec code couleur (Rouge/Orange/Vert)
- ✅ Mode collaboratif (Kanban)
- ✅ Design responsive (Bootstrap 5)
